# 社長報告：SnsClub 特進コース設計案（離脱・成功傾向分析を踏まえて）

**報告目的：** 卒業時アンケート・クレーム・卒業生投稿数分析に基づく「離脱・成功の傾向」と、特進コースとしての打ち出し方・コーチの関わり方の変更案を整理し、**どんな特進コースにするか**を具体的にまとめます。

---

## 1. エグゼクティブサマリ

| 項目 | 内容 |
|------|------|
| **特進コースの位置づけ** | **「本気で人生を変えにいくコース」**。卒業生分析から「大量投稿は万垢達成と強く相関」しており、**100投稿〜120投稿を目指して投稿を追わせるコース**として設計する。 |
| **現状の課題** | 現行コミットは「やりきるならコミット」「不安をお金で解消」の切り口で、**頑張れない人が前提で入会**してくる側面がある。特進として「結果を出す・人生を変える」と打ち出すには、**入会前の期待値調整**と**入会後の「強制的な環境」**の両方が必要。 |
| **コーチの関わり方の変更** | **前半（入会〜約3ヶ月）にコーチング・接触頻度を集中**させる。後半3ヶ月は月1回ペースにし、**全12回のうち9回を前半に前倒し**。序盤で「操作がわからない」「一人で進められない」「初投稿まで踏み切れない」を解消する。 |

---

## 2. 根拠の整理：離脱・成功分析と卒業生データ

### 2.1 離脱・成功傾向分析から得た結論（要約）

- **離脱リスク該当**：回答者の約54%（約620人）が、投稿数10以下・自己評価3以下・満足度5以下のいずれかに該当。**「行動が止まる→投稿が少ない→自己評価が下がる」**が重なって卒業まで続く。
- **離脱要因の上位**：①操作・使い方がわからない（162件）②質問・サポート不足感（138件）③一人で進める負担・孤独（112件）④ペース・時間感（105件）⑤教材・視聴の進め方（30件）。**「定期的に一緒に進める場（作業会・講義会）が欲しかった」**が③・⑤に直結。
- **成功者の特徴**：満足度・自己評価・投稿数がともに高い層（約24%）は、**「クラス制度・仲間」「講師に気軽に質問できる」「伴走・手厚いフォロー」「一人ではやめていた」**と回答。**「一人にしない」「質問できる」「仲間と一緒に進める」**が成功の中心。
- **クレームの共通軸**：入会直後の「案内が届かない・進捗が見えない」、**フィードバックが抽象的で「何をすればいいかわからない」**、講師の返信遅延・1on1の期待と現実のギャップ。→ **入会直後の見える化**と**具体的なフィードバック・応答の一貫性**が満足と継続の中心。

### 2.2 卒業生データ：投稿数と万垢達成の関係

- 卒業生のうち、**万垢達成者の合計投稿数は中央値86・平均80前後**、非達成者は中央値33・平均40前後。**大量投稿は万垢達成と強く相関**している。
- 特進コースでは、この分析を根拠に **「100投稿」または「120投稿」を目指して投稿を追わせる** 設計とする。

### 2.3 初動のスピードと継続・成果の関係

- **万垢達成者**は「とにかくすぐ始めた」「走りながら考える」傾向。**未達成者**は「初期設計に時間がかかった」「操作がわからず挫折」「投稿数0〜10で止まる」が多い。
- **「初投稿までを入会から30日以内にスタートさせる」**ことは、離脱分析（初期のつまずき・一人で進められない）と初動分析の両方に効く。

## 3. 現状のコミットと「特進」として打ち出すときの整理

- 現状のコミットは、営業が「不安をお金で解消」「やりきるならコミット」という切り口で訴求しており、**そもそも頑張れない人・結果よりやりきりを重視する人が前提**で入ってくる側面がある。
- **特進コース**を「人生を変えるコース」「結果を出すコース」として打ち出す場合、  
  - **入会前**：**「本気で人生を変えにいく人向け」「投稿をしっかり積み上げて万垢を狙う人向け」**と期待値を明確にし、選ばれるコースにする。  
  - **入会後**：**頑張れない人でも離脱させない**ために、**「やるための強制的な環境」**（毎週の作業会・毎週のコーチング・初投稿30日）をプログラムで組み込み、**学習する姿勢が不足している人にも、習慣と場で補う**設計にする。
- つまり、**特進＝「本気で人生を変える」スタンス**と、**中身は「強制的に時間と場を確保して、序盤の離脱と初動の遅れを防ぐ」**の両立が、設計の軸になる。

## 4. 特進コースの具体的な設計案

### 4.1 位置づけ・ゴール

| 項目 | 内容 |
|------|------|
| **コースのスタンス** | SnsClub 特進コースは **「本気で人生を変えにいくコース」**。 |
| **成果目標** | 卒業生傾向に基づき、**100投稿もしくは120投稿を目指して投稿をするコース**とする。 |
| **プログラムの柱** | ① 序盤（1ヶ月目）の**セッション頻度を高めてマインドと行動を整える** ② **ゴール＝行動にフォーカスするコーチング**（早く投稿に乗る・初投稿を前に出す）。 |

### 4.2 3本柱の施策（具体的）

| 施策 | 内容 |
|------|------|
| **① 入会後3ヶ月・毎週1回の作業会を強制参加** | 離脱分析で「操作がわからない」「一人で進めてやらなくなる」「講義会・作業会が欲しかった」が突出。**3ヶ月間、毎週1回、作業会に強制参加**とし、操作・使い方・「どこに何があるか」を**一緒に画面を見ながら**定期的に確認する。質問OKの時間として明示し、同じ時間に一緒に取り組むことで孤独感を減らす。 |
| **② 入会後1ヶ月・毎週1回コーチング** | 現状は月2回ペースでコンスタントに実施。特進では**1ヶ月目に限り、毎週1回コーチング**とし、**ひたすらコーチング担当者とSNSのことを考える時間を強制的に作る**。12回のうち**4回を前倒し**し、**投稿への意識を高める・初期設計や初投稿までの作業のしにくさを解消する**ことに充てる。 |
| **③ 初投稿までを入会から30日以内にスタートさせるプログラム** | 「特進コース」の必須目標として**「初投稿までを入会から30日以内にスタートさせる」**を掲げる。初動分析（即スタート型が継続率が高い）と離脱分析（初期のつまずきで離脱）の両方に対応する。 |

### 4.3 コーチングセッションの配分（前倒し設計）

| 項目 | 内容 |
|------|------|
| **現状** | コミットコースは月2回ペースで約12回を6ヶ月に均等配置。 |
| **変更案** | **前半（入会〜2〜3ヶ月終わり）にコーチングの比重を高める**。後半の残り3ヶ月は**月1回ペース**とし、**12回のうち9回を前半に前倒し**する形で配分する。 |
| **オリエンテーション** | オリエンテーションは**オリエンテーション専用**とし、そのうえで**前半に、自己分析を深めた上での目標設定・アカウントを学んだ上での目標設定**を行う流れに全体を組み替える。 |
| **意図** | 離脱・成功傾向にある人の習慣（仲間と進める・質問できる・伴走）を**序盤で取り入れた期間**にし、前半の離脱を防ぎ、初動を早める。 |

### 4.4 学習マインドセット・強制的な環境

- **課題認識**：成果が出るまで下手で楽しくないからやめてしまう。**やる気はやらないと出てこない**が、「やるための強制的な環境」が不足しているから行動しない。また、学習しに来ているのに**学習する姿勢が不足している**ケースがある。
- **特進での対応**：**強制的に時間と場を確保する**（毎週作業会・毎週コーチング・初投稿30日）。「何をしていてもよいが、一緒に時間を確保する」だけでも、一人で教材を進めてやらなくなる層を減らす（離脱分析の「教材を1人で進めるのが大変」に応える）。

## 5. 離脱・成功分析との対応一覧（施策の根拠）

特進コースの各施策が、分析のどの課題に効くかを一覧にしました。

| 分析で見えた課題 | 特進コースの施策 |
|------------------|------------------|
| ① 操作・使い方がわからない（162件） | 入会後3ヶ月・毎週1回の**作業会で一緒に画面を見ながら**操作・Portal・Discord・提出先を確認。 |
| ② 質問・サポート不足感（138件） | 作業会を**「質問OKの時間」**として明示。質問の入口と返信期待値（○日以内等）を案内し続ける。 |
| ③ 一人で進める負担・孤独（112件） | **毎週同じ時間に一緒に取り組む**作業会の強制参加。クラス単位の講義会・作業会ニーズに応える。 |
| ④ ペース・時間感（105件） | 前半に**セッション頻度を高め**、ペースと「何をいつまでにやるか」をコーチングで明確にする。初投稿30日で**最初のマイルストーン**を共有。 |
| ⑤ 教材・視聴の進め方（30件） | 作業会で「今月の使い方・よくあるつまずき」を短時間共有。進捗の見える化で止まっている人を早めに検知。 |
| クレーム：初期の見えない・わからない | 入会直後の**案内・進捗の見える化**は既存の改善として継続。作業会で**繰り返し仕組みを説明**する。 |
| クレーム：フィードバックが抽象的 | 講師・コーチの**フィードバックの型**（何をどう直すかを1つ書く・見本添付）の標準化は、特進に限らず推進。作業会で**よくあるつまずきを直接説明**する場としても使う。 |
| 成功要因：仲間・質問・伴走 | **作業会で「仲間と一緒に進める」**、**毎週コーチングで伴走**。前半に接触頻度を集中させることで再現する。 |
| 初動が遅いと継続率低下 | **「初投稿まで30日」**をプログラムの必須目標にし、初期設計・初投稿の壁を**毎週コーチングと作業会**で解消する。 |

## 6. 期待効果と次のステップ

### 6.1 期待できる効果

1. **操作・使い方での離脱**：毎週の作業会で一緒に確認するため、技術的つまずきでの離脱を減らす。
2. **質問ができずに止まる離脱**：質問OKの時間とチャネルを明示し、フィードバック待ち以外の経路を増やす。
3. **一人で進めてやらなくなる離脱**：同じ時間に一緒に取り組む習慣を作り、孤独感を減らす。
4. **初動の遅れ**：初投稿30日・毎週コーチングで、初投稿までの壁と初期設計のしにくさを解消し、継続率と投稿量の底上げを狙う。
5. **万垢に向けた投稿量**：100〜120投稿を目標にした設計と、前半の習慣づけで、卒業生分析で見えた「大量投稿→万垢」の傾向に乗せる。

### 6.2 社長判断・次のステップの提案

| 項目 | 提案 |
|------|------|
| **特進コースの定義** | 「本気で人生を変えにいくコース」とし、**100投稿〜120投稿を目指す**ことを明文化する。 |
| **営業・期待値調整** | 「結果を出す・人生を変える」と打ち出す場合、**入会前の期待値**（本気で投稿を積み上げる人向け）を営業トーク・募集要項で明確にする。 |
| **プログラム改修** | ① 入会後3ヶ月・毎週1回作業会の強制参加 ② 入会後1ヶ月・毎週1回コーチング ③ 初投稿まで30日以内スタート ④ セッション配分の前倒し（前半9回・後半月1回）と、オリエンテーション・目標設定の流れの見直し。 |
| **運営・講師側** | 入会直後の案内・進捗の見える化、フィードバックの具体化・標準化は、特進に限らず継続して実施する。 |
